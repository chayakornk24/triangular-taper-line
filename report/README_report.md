# Report (PDF)

This folder contains the final step-by-step project report
for the triangular taper transmission line.

## File
- `Triangular_Taper_Line_Report.pdf`
  - Complete report including:
    - Design workflow
    - Netlist simulation
    - Geometry simulation (original vs expanded box cell)
    - PCB design and fabrication
    - VNA measurement setup and results
    - Simulation vs measurement comparison
    - Summary

## Notes

- Supporting project files are stored in the main repository folders:
  - `simulation/`
  - `measurement/`
  - `pcb/`
